## How to Clone:
- Copy repository or download the code
- npm install
- npm run dev

## **Tech Stack**
- React Typescript sebagai library
- Tailwind untuk styling dan layouting
- API dari *https://developer.themoviedb.org/reference/intro/getting-started*

### **Library Tambahan**
- Axios
- React Router
- React Icons
- React Toastify
- React Slick (Carousel)
